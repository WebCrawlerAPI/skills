# webcrawlerapi-integration skill

A Claude Code skill that helps integrate [WebCrawlerAPI](https://webcrawlerapi.com) into any codebase.

## What it does

When triggered, loads full integration context into Claude's session — API endpoints, SDK usage (JS/Python), output formats, error codes, and links to docs.

## Triggers

Activates when you ask things like:
- "integrate webcrawlerapi into my project"
- "scrape a website with webcrawlerapi"
- "add web scraping to my app"
- "crawl pages using webcrawlerapi"
- "set up webcrawlerapi"
is 
## Covers

- Authentication setup
- Single-page scraping (`POST /v2/scrape`)
- Multi-page crawling (`POST /v1/crawl`)
- JS/TS and Python SDK examples (sync + async)
- Output formats: `markdown`, `cleaned`, `html`, `links`
- Structured data extraction with AI + JSON schema
- Webhooks and async patterns
- Website change monitoring (Feeds)
- Error codes and rate limits

## Docs

- Full docs: https://webcrawlerapi.com/docs
- API reference: https://webcrawlerapi.com/docs/api
